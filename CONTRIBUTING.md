# Contributing
You can submit PRs [here](https://github.com/averagebagelenjoyer/JShell/pulls),  
and issues [here](https://github.com/averagebagelenjoyer/JShell/issues).

Be very clear, the exact purpose of a given pull request, or the exact issue and how to recreate it.  
Just don't be a jackass, I'll read over the request and likely respond.

I have the ability to decline a pull request on command, so please, understand the project and it's purpose before attempting to contribute to it—for everyone's sake.
